# Federated Fetal Health Classification

An independent ML project using a public Kaggle healthcare dataset.
This repo starts with a clean centralized baseline (Scenario 1) and will later extend to GAN augmentation and Federated Learning.

## Dataset
Source (Kaggle):
https://www.kaggle.com/datasets/andrewmvd/fetal-health-classification

## Scenario 1 (Baseline)
- Multinomial Logistic Regression (interpretable baseline)
- LightGBM (high-performance baseline)

## How to Run
```bash
cd src
python scenario_1_baseline.py
