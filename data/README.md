# Dataset Information

This project uses the **Fetal Health Classification** dataset,
publicly available on Kaggle.

Source:
https://www.kaggle.com/datasets/andrewmvd/fetal-health-classification

## Description
The dataset contains cardiotocography (CTG) features used to classify fetal health status into three categories:
- Normal
- Suspect
- Pathological

## Notes
This dataset is public and used here for educational/research demonstration.
